# Contributing

Thanks for your interest in improving The Tablets of Stone! This is a small project meant to be useful for Primary teachers, parents, and anyone teaching the Ten Commandments. Contributions of all sizes are welcome.

## Easy Ways to Help

**You don't have to be a developer to contribute.** Here are some things anyone can help with:

- **Suggest better stories or scriptures.** If you know a more powerful or more kid-friendly scripture for one of the commandments, open an issue and tell us about it.
- **Report bugs or confusing wording.** If something doesn't work right or a story summary feels off, let us know.
- **Share how you used it.** Stories about how the game went in your Primary class help us improve it.

To do any of these, just [open an issue](../../issues/new) on GitHub.

## Editing the Game Content

All the stories, scripture references, and commandment text live in `index.html` in the `ROUNDS` and `COMMANDMENTS` arrays. You can edit them in any text editor — no build step, no compiling. Just save and reload the page in your browser.

## Submitting a Pull Request

If you're comfortable with Git:

1. Fork the repo.
2. Create a branch with a descriptive name (e.g. `improve-cain-and-abel-summary`).
3. Make your changes to `index.html` (or other files).
4. Test locally by opening `index.html` in a browser and playing through.
5. Commit and push, then open a pull request describing what you changed and why.

## Style Guidelines

- Story summaries should be 2–4 sentences, written for an 11-year-old reading level.
- Scripture quotations should match the official LDS edition exactly (KJV for the Bible).
- Keep the tone reverent but engaging — this is a game, but it's about sacred topics.
- Don't introduce external libraries or build tools. The single-file simplicity is the point.

## Questions?

Open an issue and ask. There's no such thing as a dumb question here.
